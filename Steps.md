### Setup project
1. ``yarn add --dev hardhat``
2. Set hardhat.config.js
3. Add prettier (optional)
4. Add package.json and yarn.lock
5. ``yarn add --dev @nomiclabs/hardhat-ethers@npm:hardhat-deploy-ethers ethers @nomiclabs/hardhat-etherscan @nomiclabs/hardhat-waffle chai ethereum-waffle hardhat hardhat-contract-sizer hardhat-deploy hardhat-gas-reporter prettier prettier-plugin-solidity solhint solidity-coverage dotenv``
6. Add .env
7. Add openZeppelin
8. Add helper-hardhat-config.js
9. Write smart contract and deploying scripts
